##### Quy trình tải lên sản phẩm từ file excel

<br />

- Bước 1: Vào trang Danh sách sản phẩm
  - Vào trang quản trị SKYCOM, đi tới Sản phẩm <ArrowRightAltIcon sx={{verticalAlign: "bottom"}} /> Danh mục sản phẩm. Tại mục Sản phẩm nhấp vào nút **“Nhập file”** để tiến hành tải lên nhiều sản phẩm.
- Bước 2: Nhập file sản phẩm

  - Sau khi nhấp nút **“Nhập file”** pop-up Nhập file hiện lên, bạn tiến hành nhập file và điền các thông tin:
  - Nhập file: Tải file lên theo mẫu có sẵn, bạn có thể lấy file mẫu bằng cách nhấn nút **“File mẫu”**

  > Lưu ý: File tải lên phải đảm bảo các điều kiện sau: Trường bắt buộc không được để trống - trường bắt buộc được ký hiệu có dấu “ \* ”.
  > Hệ thống chỉ ghi nhận file sản phẩm là file Excel (.xls) và file CSV (.cvs)

  - Click chọn **“Choose file”** để chọn file có chứa danh sách sản phẩm từ máy của bạn.
  - File tải lên thành công sẽ hiển thị thông tin tên file Excel hoặc file csv.
  - Chọn giá trị cột:
  - Chỉnh sửa ô
  - Click chọn “Thêm” tải file lên.

- File tải lên thành công sẽ hiện thông báo **“Tải lên thành công”**
