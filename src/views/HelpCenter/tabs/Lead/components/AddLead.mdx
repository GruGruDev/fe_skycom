##### Để tạo danh sách khách hàng tiềm năng từ file dữ liệu có sẵn vui lòng làm theo hướng dẫn sau:

<br />

> Lưu ý: Tài khoản được quản trị cấp quyền Tạo Lead bằng file mới có thể Tạo danh sách khách hàng tiềm năng từ upload file.

- B1: Truy cập vào **Lead Center** -> Danh sách **Lead Center** trong thanh menu
- B2: Click chọn **“Import Excel"** để tải data từ file Excel lên hệ thống
- B3: Trong form Import Excel bạn thực hiện:

  - Nhập file bằng cách nhấp vào **“Choose File”** để chọn file từ máy tính

    > Lưu ý: File dữ liệu phải có cấu trúc giống như file mẫu (bạn có thể lấy file mẫu bằng cách nhấp vào nút **“File mẫu”**)

- B3: Chọn giá trị cột:
  - **Kênh bán hàng (bắt buộc):** bạn bắt buộc chọn kênh bán hàng cho file dữ liệu. Kênh bán hàng được chọn sẽ áp dụng cho toàn bộ dữ liệu trong file
  - **Sản phẩm:** Chọn sản phẩm cho file dữ liệu. Sản phẩm được chọn sẽ được áp dụng cho tất cả dữ liệu trong file
  - **Phòng kinh doanh:** Khi bạn chọn Phòng kinh doanh thì file dữ liệu được tải lên sẽ mặc định chia về phòng kinh doanh đó.
  - **Ghi chú:** Có thể có hoặc không
- B4: Nhấp vào nút **Thêm** để tiến hành tải dữ liệu lên hệ thống từ file chứa dữ liệu khách hàng tiềm năng
