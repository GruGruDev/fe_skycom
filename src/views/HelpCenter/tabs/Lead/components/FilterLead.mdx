##### Để lọc, tìm kiếm và xuất file khách hàng tiềm năng hãy làm theo hướng dẫn bên dưới

<br />

Bạn có thể dễ dàng tìm kiếm và lọc khách hàng tiềm năng theo Trạng thái khách hàng tiềm năng, nhãn tùy chỉnh, người nhận số và các tham số khác trong khoảng thời gian chỉ định.

- Lọc khách hàng tiềm năng theo thuộc tính:
  - ​Thời gian: Nhấp vào Filter <ArrowRightAltIcon sx={{verticalAlign: "bottom"}} /> Thời gian tạo/Thời gian chia số/Thời gian hoàn thành/Ngày gọi lại
    <ArrowRightAltIcon sx={{ verticalAlign: "bottom" }} /> Chọn thời gian bạn muốn lọc
  - Người: Nhấp vào Filter <ArrowRightAltIcon sx={{verticalAlign: "bottom"}} /> Người nhận số/Người tạo
    <ArrowRightAltIcon sx={{ verticalAlign: "bottom" }} /> Chọn người bạn muốn lọc
  - Khác: Nhấp vào FIlter <ArrowRightAltIcon sx={{verticalAlign: "bottom"}} /> Kênh bán hàng/Sản phẩm/Phòng kinh doanh/Phòng ban/Số cuộc gọi/Lý do Đang xử lý/Lý do không mua/Lý do Dữ liệu KCL
    <ArrowRightAltIcon sx={{ verticalAlign: "bottom" }} /> Chọn thẻ tương ứng bạn muốn lọc
- Lọc khách hàng tiềm năng theo trạng thái:
  - ​Nhấp vào nút Filter <ArrowRightAltIcon sx={{verticalAlign: "bottom"}} /> Trạng thái Lead
    <ArrowRightAltIcon sx={{ verticalAlign: "bottom" }} /> Chọn trạng thái bạn muốn lọc
- Tìm kiếm khách hàng tiềm năng thông qua những đặc điểm nhận dạng sau:
  - Tên khách hàng
  - Số điện thoại khách hàng
- Xuất file khách hàng tiềm năng: Bạn chỉ có thể Xuất file khi được quản trị viên cấp quyền Xuất file qua email. File xuất về sẽ được lưu về email đã đăng ký tài khoản.
